As Principais Linguagens De Programação

Linguagem de programação é o idioma que você vai utilizar para comunicar ao computador e escrever os códigos. Assim como na comunicação entre humanos, você emite uma mensagem e precisa que seu receptor a entenda e responda.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Java

Java é o ambiente computacional, ou plataforma, criada pela empresa estadunidense Sun Microsystems, e vendida para a Oracle depois de alguns anos. A plataforma permite desenvolver programas utilizando a linguagem de programação Java.

PHP

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.

Leia Mais

Python

Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.

Leia Mais

C++

C++ é uma linguagem de programação compilada multi-paradigma e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores.

C

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix.

Notícias que chamaram a nossa atenção :

Célula de biocombustível utiliza suor humano para gerar energia: o dispositivo, desenvolvido por cientistas japoneses, gera energia elétrica a partir do lactato presente no suor, abrindo a possibilidade de aparelhos eletrônicos vestíveis como relógios inteligentes que não precisam de baterias. Os pesquisadores conseguiram manter ligado um medidor de atividades comercialmente disponível por uma hora e meia utilizando apenas uma gota de suor artificial. As informações são da Tokyo University of Science.

Smartphone que se transforma em tablet: o “Fold ‘n Roll” da TCL pode ser utilizado como um celular normal com uma tela de 6,8 polegadas, se desdobrar para 8,8 polegadas, e ainda expandir a tela enrolável para 10 polegadas. A TCL afirma que ainda está definindo as especificações técnicas do aparelho e não forneceu uma data para o lançamento. As informações são do site The Verge.

NASA vai lançar enxame de “pipas” cheias de sensores na atmosfera de Vênus: cada pipa do projeto “LEAVES” pesa 130g e voará por cerca de 9 horas coletando dados importantes sobre o estado dinâmico e a composição da atmosfera do planeta. As pipas serão lançadas de um satélite que ficará orbitando Vênus, reduzindo assim os custos e riscos da missão. As informações são da página da NASA.

Projeto de Lei cria regras para remoção de conteúdo de sites e redes sociais: a finalidade do PL 649/21 é exigir maior transparência e mecanismos que possibilitem o exercício da ampla defesa por usuários que se sentirem prejudicados com a remoção de suas postagens. As informações são da Agência Câmara Notícias.

PIX supera marca de 1 bilhão de transações: desde o lançamento, em novembro do ano passado, o sistema de pagamentos instantâneo já transacionou 787 bilhões de reais. Já são 75,6 milhões de pessoas físicas cadastradas no PIX e 5 milhões de empresas. As informações são do site Mobile Time.
